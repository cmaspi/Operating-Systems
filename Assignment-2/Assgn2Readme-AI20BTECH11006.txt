## How to RUN?
1. Make sure you have the `input.txt` file, if not create it.
2. Enter suitable numbers for $N$ and $K$ in the `input.txt`.
3. Use the following commands to run the code

Compilation
-----------
```c
gcc Assgn2Src-AI20BTECH11006.c -o Assgn2Src-AI20BTECH11006.out
```

Running
-------
```c
./Assgn2Src-AI20BTECH11006.out
```

The output generated will be stored in the same directory as the executable file.
